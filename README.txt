TELEGRAM QUIZ BOT - Инструкция по установке

ФАЙЛЫ ПРОЕКТА:
- main.py - точка входа в приложение
- config.py - конфигурация и настройки
- aiogram_bot.py - основная логика бота
- requirements.txt - зависимости Python

УСТАНОВКА:
1. Убедитесь, что установлен Python 3.8+
2. Установите зависимости: pip install -r requirements.txt
3. Создайте переменную окружения TELEGRAM_BOT_TOKEN со значением вашего токена
   или используйте токен по умолчанию из файла config.py
4. Запустите бота: python main.py
